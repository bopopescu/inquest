Readme: 

To run the front end, you need following files on top of the bottle framework 
inquest.py
static/inquest.css
static/resultspage.css 
static/search.jpg

to run the front end 
1) On the command line, run "python inquest.py"
2) On the browser, go to http://localhost:8080/

To run the backend, you need the following file
1) crawler.py (Modified) 

To test the back end, you need 'test_crawler.py' to be in same folder as crawler.py
1) On the command line, run 'python test_crawler.py'
